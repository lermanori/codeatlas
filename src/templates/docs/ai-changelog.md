# Changelog

This document tracks changes made to the project, especially those made with AI assistance.

## Format

Each entry should follow this format:

```markdown
## YYYY-MM-DD

### Changed
- [List of files changed]
- [Brief description of changes]

### Added
- [New features or files added]

### Fixed
- [Bugs fixed]

### Notes
[Any additional notes or context]
```

## Example Entry

## 2024-01-15

### Changed
- `src/api/auth.ts` - Updated authentication flow to use JWT tokens
- `modules/auth.md` - Updated module documentation to reflect JWT changes

### Added
- `src/utils/jwt.ts` - New JWT utility functions

### Fixed
- Fixed token expiration handling in authentication middleware

### Notes
This change improves security by using stateless JWT tokens instead of session-based authentication.
