# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-XX

### Added

- Initial release of CodeAtlas
- `ai-docs init` command to initialize documentation structure
- `ai-docs add-module` command to add new modules interactively
- `ai-docs scan` command to build documentation tree
- `ai-docs dev` command to start local documentation viewer
- AI-powered module detection and documentation generation
- Token usage tracking for all LLM operations
- Cursor IDE rule integration
- Local web-based viewer for documentation tree and token usage
- Support for Markdown files with YAML frontmatter
- Automated template file generation
- Interactive CLI with prompts for module creation

### Features

- Structured documentation system optimized for AI assistants
- Automatic module suggestion based on codebase structure
- Documentation tree visualization
- Token usage statistics and tracking
- Cursor rule for automatic documentation updates
